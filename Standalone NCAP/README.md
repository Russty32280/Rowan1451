# Stand Alone Python Based NCAP

## Dependencies
- Paho MQTT client

# NCAP ID
There are two options for you to pick for your NCAP ID. You can provide an NCAP ID or you can allow a random ID to be generated.

RUSMARTLAB/Heartbeat

RUSMARTLAB/NCAP_ID
## NCAP --> Application
RUSMARTLAB/APPL_ID
